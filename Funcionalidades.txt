FUNCIONALIDADES POR REBANADAS
1. TOMA EN CUENTA LA CADENA VACÍA: Para una cadena "" -> 0
2. TOMA EN CUENTA UNA CADENA CON UN SOLO NÚMERO: Para una cadena "2"-> 2 
3. TOMA EN CUENTA UNA CADENA CON DOS NÚMEROS SUMADOS POR UNA COMA: Para una cadena "1,2" -> 3 
4. TOMA EN CUENTA UNA CADENA CON VARIOS NÚMEROS: Para una cadena "1,2,3" -> 6
5. TOMA EN CUENTA UNA CADENA SEPARADA POR GUIONES: Para una cadena "1,2-3,4" -> 10
6. RECONOCE UN SEPARADOR DETERMINADO POR EL USUARIO: Para una cadena "//[;] 1,2;3-4" -> 10
7. RECONOCE SEPARADORES MAYORES A UN CARACTER: Para una cadena "//[.-] 1.-2.-3.-4" -> 10
8. SE RECONOCE MÁS DE UN SEPARADOR BAJO EL MISMO FORMATO "//[*][$] 1$2*,3-4" -> 10
9. SI UN NUMERO EN LA CADENA ES MAYOR A 1000 ES IGNORADO: Para una cadena "1000,1001" -> 1000
